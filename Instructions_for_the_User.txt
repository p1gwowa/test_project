Інструкція для Користувача Додатку "Прогнозування Відтоку Клієнтів для Телекомунікаційної Компанії" (Хмарне Середовище)
1. Опис Додатку

Цей додаток розгорнутий у хмарному середовищі та призначений для прогнозування ймовірності відтоку клієнтів телекомунікаційної компанії. Користувачі можуть вводити дані про одного клієнта вручну або завантажувати файл у форматі CSV для масового аналізу.

2. Використання Додатку

Доступ до Додатку:
Перейдіть за посиланням до хмарного додатку через веб-браузер.
Авторизуйтеся, якщо це необхідно.
Введення Даних для Одиничного Клієнта:
У спеціальній формі введіть дані про клієнта:
is_tv_subscriber (0 або 1)
is_movie_package_subscriber (0 або 1)
subscription_age (у місяцях)
reamining_contract (у місяцях)
download_avg (середня швидкість завантаження)
upload_avg (середня швидкість вивантаження)
download_over_limit (0 або 1)
Натисніть кнопку для запуску прогнозу.
Завантаження CSV-файлу для Масового Аналізу:
Завантажте файл у форматі CSV, що містить дані про кількох клієнтів, через відповідний інтерфейс.
Дані повинні відповідати наступній структурі колонок:
is_tv_subscriber, is_movie_package_subscriber, subscription_age, reamining_contract, download_avg, upload_avg, download_over_limit.
Після завантаження система автоматично виконає прогноз для всіх клієнтів з файлу.

3. Валідація Даних

Для забезпечення коректності прогнозування, введені дані проходять валідацію:

Числові поля:
subscription_age, reamining_contract, download_avg, upload_avg: перевіряються на наявність числових значень. Вони не можуть бути від’ємними.
Бінарні поля:
is_tv_subscriber, is_movie_package_subscriber, download_over_limit: мають приймати значення 0 або 1.
Цілісність Даних:
Перевіряється наявність всіх необхідних полів. У випадку введення неповних або некоректних даних, система видасть відповідне повідомлення про помилку та запропонує виправити введення.

4. Аналіз та Візуалізація

Результати Прогнозування:
Для кожного клієнта (як введеного вручну, так і завантаженого через файл) система надає прогноз ймовірності відтоку та рекомендації.

Візуалізація:
Додаток також генерує графіки для кращого розуміння впливу різних факторів на ймовірність відтоку.

5. Завершення Роботи ???

Після завершення роботи користувач може завантажити результати прогнозу, що включають ймовірності відтоку та візуалізації, у форматі CSV або PDF для подальшого використання.

Цей додаток є потужним інструментом для аналізу клієнтських даних та запобігання відтоку клієнтів у телекомунікаційній сфері.
